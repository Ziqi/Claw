### 高价值目标攻击记录

#### 目标 1: TP-Link TL-WDR5620 (10.130.1.252)

| 阶段 | 操作 | 结果 |
|---|---|---|
| 信息收集 | HTTP 指纹确认型号 + `/ds` 入口探测 | ✅ 确认 TL-WDR5620 千兆易展版 |
| CVE 验证 | CVE-2024-46486 `method:get` 探测 | HTTP 401 (鉴权生效) |
| 精准验证 | Plan A: `method:do` + `cfgsync` 旁路 | HTTP 401 (已修补) |
| 固件分析 | Plan B: 官方固件下载 | V2.0 已下架，仅存 V4/V6/V9 |
| **结论** | **路由器固件已升级，CVE-2024-46486 不可利用** | 🟢 安全 |

> **学习价值**：完整实践了从 CVE 研究 → PoC 编写 → 实战验证 → 固件逆向的漏洞研究全流程
